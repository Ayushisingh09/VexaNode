# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in VexaNode, **do not open a public GitHub issue.**

Report it privately by contacting the maintainers directly via the [VexaNode Discord](https://discord.gg/vexanode) or by emailing the team.

Please include:
- A clear description of the vulnerability
- Steps to reproduce it
- Potential impact
- Any suggested fix if you have one

We will acknowledge the report within 48 hours and aim to release a fix as quickly as possible. You will be credited in the changelog if you wish.

## Supported Versions

| Version | Supported |
|---|---|
| `main` branch | Yes |
| Older forks / branches | No |
