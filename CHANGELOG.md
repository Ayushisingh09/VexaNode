## [1.0.0] — 2026-06-07

### Added
- Next.js 14 App Router storefront with glassmorphic dark-theme design
- VPS, game server, Lavalink, web hosting, and domains pages
- Billing frequency calculator with monthly/annual toggle and 10% discount
- NextAuth v4 authentication — Google OAuth, Discord OAuth, and credentials-based login
- Bcrypt password hashing (salt rounds: 10)
- Client dashboard — active orders, invoices, and payment proof upload
- Support ticket system with priority states and reply threads
- Admin panel — user directory, order management, ticket queue, and global config controls
- Cashfree payment gateway integration
- Supabase PostgreSQL backend with automatic profile creation trigger
- Domain availability checker
